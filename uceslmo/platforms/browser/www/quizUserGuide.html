Quiz App User Guide

The Quiz App

The Quiz App for UCL campus gives the user questions to answer as they walk around Bloomsbury and the UCL campus.

How do I use the Quiz App?

1. First, the user should click the 'Get Questions Points' button to add the quiz question locations to the map. The button can be found under the main menu on the left. The main menu may be hidden but can be revealed by clicking the menu button, which is symbolised by three horizontal lines.

2. The user should walk around Bloomsbury, tracking their location by clicking on the 'Track User Location' button, which can also be found under the main menu on the left.

3. When the user approaches a location with a quiz question, the user can enter their answer choice in the form below the map. To check if the answer chosen is correct, click on the 'Check if you are correct!' button, and the correct answer should appear beneath.
